## PPT / Keynote Export

| Cách | Phù hợp khi |
|------|-------------|
| Screenshot tĩnh | Cần nhanh, slide không cần animation |
| Screen record → GIF/MP4 | Muốn giữ animation trong PPT |
| Google Slides + HTML embed | Audience xem online |
| Copy CSS animation → Keynote | Keynote hỗ trợ một số CSS transition |

Để export: mở component file → Cmd+Shift+4 (screenshot) hoặc QuickTime record.
